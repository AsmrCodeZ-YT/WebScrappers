
# ابزار استخراج آگهی‌های شغلی JobVision

قبل از هرچیز ااین نکته رو میخوام بهش اشاره کنم :این پروژه صرفاً برای اهداف آموزشی و شخصی ارائه شده است. لطفاً قوانین و شرایط استفاده از وب‌سایت JobVision را رعایت نمایید. 

این پروژه یک اسکریپت پایتون برای دریافت آگهی‌های شغلی از وب‌سایت [jobvision.ir](https://jobvision.ir) با استفاده از API داخلی آن است. خروجی به‌صورت فایل‌های CSV ذخیره می‌شود و از پردازش هم‌زمان (async) برای سرعت بیشتر بهره می‌برد.

البته ۲ نسخه ازش رو گذاشتم اون یکی کمی ساده تر هست و به صورت مرحله ای جلو میره غیر اسینک هست.

## ویژگی‌ها

- دریافت خودکار **همه آگهی‌ها** در یک دسته‌بندی شغلی خاص (مثلاً `data-science`).
- امکان انتخاب **بازه زمانی** (همه زمان‌ها، ۳ روز، ۷ روز، ۱۵ روز، ۳۰ روز گذشته).
- دریافت **جزئیات کامل** ۲۰ آگهی اول و ذخیره جداگانه.
- نمایش لاگ هم در کنسول و هم در فایل `scraper.log` برای رهگیری.
- تبدیل خروجی JSON تو در تو به ساختار مسطح (flat) و ذخیره در CSV.
- استفاده از درخواست‌های غیرهمزمان برای عملکرد سریع.

## پیش‌نیازها

- Python نسخه ۳.۷ یا بالاتر
- کتابخانه‌های زیر (نصب با دستور `pip install`):

| کتابخانه       | کاربرد                     |
|----------------|----------------------------|
| aiohttp        | درخواست‌های HTTP غیرهمزمان |
| pandas         | ساخت DataFrame و ذخیره CSV |
| beautifulsoup4 | تبدیل HTML به متن ساده     |

نصب یک‌جا:

```bash
pip install aiohttp pandas beautifulsoup4
```

## راهنمای استفاده

### ۱. دریافت اسکریپت

فایل اسکریپت را دانلود کنید یا با `git clone` دریافت نمایید. (فایل `scraper.py`)

### ۲. نصب کتابخانه‌ها

دستور فوق را در ترمینال اجرا کنید تا وابستگی‌ها نصب شوند.

### ۳. اجرای اسکریپت

```bash
python aysinc_job_vision.py

python Job_vision.py # or use this
```

### ۴. وارد کردن آدرس دسته‌بندی

برنامه از شما یک آدرس شبیه نمونه زیر می‌خواهد:

```
https://jobvision.ir/jobs/category/data-science?page=1&sort=0
```

**توجه:** فقط بخش `category/دسته‌بندی` استخراج می‌شود. بنابراین می‌توانید هر آدرسی از یک جستجوی معتبر در سایت بدهید، مثلاً:

```
https://jobvision.ir/jobs/category/front-end?page=1&sort=0
```

### ۵. انتخاب بازه زمانی

بعد از آن، منویی با گزینه‌های زیر نمایش داده می‌شود:

```
Available time ranges:
  0: ALL
  1: 3 Days Ago
  2: 7 Days Ago
  3: 15 Days Ago
  4: 30 Days Ago
Enter time range number:
```

یکی از اعداد ۰ تا ۴ را وارد کنید. برای دریافت همه آگهی‌ها بدون محدودیت زمانی، `0` را انتخاب کنید.

### ۶. پردازش و دریافت خروجی

اسکریپت شروع به کار می‌کند و مراحل زیر را طی می‌کند:

1. دریافت تعداد کل صفحات آگهی
2. دریافت هم‌زمان تمام صفحات و ذخیره در فایل CSV اول (مثلاً `data-science_11-06-2026.csv`)
3. دریافت جزئیات ۲۰ آگهی اول و ذخیره در فایل CSV دوم (مثلاً `data-science_11-06-2026_BYID.csv`)

فایل لاگ `scraper.log` نیز در کنار اسکریپت ایجاد می‌شود.


## تنظیمات و شخصی‌سازی

در بخش ثابت‌های اسکریپت می‌توانید مقادیر زیر را تغییر دهید:

| متغیر                | مقدار پیش‌فرض | توضیح                                |
|----------------------|---------------|--------------------------------------|
| `PAGE_SIZE`          | 30            | تعداد آگهی در هر صفحه (API از ۳۰ پشتیبانی می‌کند) |
| `DETAIL_CONCURRENCY` | 10            | حداکثر درخواست‌های هم‌زمان جزئیات    |
| `TIME_RANGES`        | دیکشنری از قبل تعریف شده | می‌توانید بازه‌های زمانی بیشتری اضافه کنید* |
| `BASE_URL`           | https://jobvision.ir | آدرس اصلی سایت (برای دریافت کوکی اولیه) |

*توجه: مقادیر عددی بازه زمانی باید با API هماهنگ باشد. در حال حاضر مقادیر ۰ تا ۴ معتبر هستند.

## لاگ‌ها

تمام رویدادها و خطاها هم در کنسول و هم در فایل `scraper.log` ثبت می‌شوند. فایل لاگ با هر بار اجرا بازنویسی نمی‌شود بلکه اطلاعات جدید به انتهای آن اضافه می‌شود (FileHandler بدون mode='w'). اگر بخواهید لاگ قبلی پاک شود، فایل را پیش از اجرا حذف کنید.

## نکات مهم

- **احترام به سرور:** اگر خطای HTTP 4xx/5xx دریافت کنید، اسکریپت آن را لاگ کرده و رد می‌شود. تعداد درخواست‌های هم‌زمان عمداً محدود شده است.
- **تغییرات API:** اندپوینت‌ها و ساختار پاسخ API ممکن است در آینده تغییر کنند. در صورت بروز خطا، فایل لاگ را بررسی کنید و در صورت نیاز مسیرهای API را به‌روز کنید.
- **متوقف کردن:** با فشردن `Ctrl+C` می‌توانید اجرا را قطع کنید. اسکریپت این وقفه را مدیریت کرده و پیام مناسبی نشان می‌دهد.
- **پشتیبانی از دسته‌بندی‌های دیگر:** می‌توانید هر دسته‌بندی معتبری را از URL استخراج کنید، مانند `front-end`، `back-end`، `mobile` و غیره.

## رفع اشکال رایج

- **خطای `Could not extract job category from URL`**: مطمئن شوید آدرس شامل `category/...` باشد. مثال صحیح: `https://jobvision.ir/jobs/category/devops?page=1&sort=0`.
- **عدم ذخیره فایل خروجی**: دسترسی نوشتن در پوشه جاری را بررسی کنید. لاگ‌ها را برای خطاهای HTTP ببینید.
- **کندی اجرا**: اگر اینترنت ضعیفی دارید، می‌توانید `DETAIL_CONCURRENCY` یا Semaphore دریافت صفحات (در حال حاضر ۵) را کاهش دهید.
